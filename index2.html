CSS Flexbox布局练习

尝试使用Flexbox属性创建一个导航菜单

flexbox-practice.html
<!DOCTYPE html> <html> <head> <style> /* 在此编写CSS代码 */ .nav-container { /* 设置容器为Flex布局 */ display: flex; /* 设置主轴方向为水平 */ flex-direction: row; /* 项目在主轴上的对齐方式 */ justify-content: space-around; /* 项目在交叉轴上的对齐方式 */ align-items: center; background: #4361ee; padding: 1rem; border-radius: 8px; } .nav-item { color: white; padding: 0.5rem 1rem; text-decoration: none; font-weight: 600; transition: all 0.3s; } .nav-item:hover { background: rgba(255, 255, 255, 0.2); border-radius: 4px; } </style> </head> <body> <div class="nav-container"> <a href="#" class="nav-item">首页</a> <a href="#" class="nav-item">课程</a> <a href="#" class="nav-item">项目</a> <a href="#" class="nav-item">社区</a> <a href="#" class="nav-item">关于</a> </div> </body> </html>

练习任务

尝试修改以下属性并观察效果：

  • 将flex-direction改为column
  • 将justify-content改为flex-start
  • 将align-items改为flex-start
  • 添加flex-wrap: wrap属性

学习提示

使用Flexbox时，记住容器属性控制所有项目的布局，而项目属性可以覆盖个别项目的默认行为。
